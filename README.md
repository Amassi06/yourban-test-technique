# Test Technique Yourban

Application web full-stack développée dans le cadre du test technique Yourban.

## Stack Technique

* **Frontend :** React 19, TypeScript, Vite
* **Backend :** Node.js, Express, TypeScript.
* **Base de données :** `box-office-200.json`

## Fonctionnalités implémentées
- [x] **Affichage des films** 
- [x] **Statistiques dynamiques** 
- [x] **Filtrage & Tri** 
- [x] **Détail d'un film** 
- [x] **CRUD Complet**
